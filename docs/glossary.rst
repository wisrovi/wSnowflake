Glossary
========

.. glossary::
   :sorted:

   Snowflake
      Cloud data platform providing data warehousing and analytics services.

   Warehouse
      Compute resource in Snowflake for processing queries.

   Database
      Logical container for schemas and objects in Snowflake.

   Schema
      Collection of database objects (tables, views, etc.) in Snowflake.

   Account
      Snowflake account identifier for authentication and resource organization.

   ORM
      Object-Relational Mapping. Technique mapping database tables to Python objects.

   Connection Manager
      Class managing database connections and session lifecycle.

   Query Builder
      Fluent interface for constructing SQL queries programmatically.

   Table Sync
      Feature synchronizing Pydantic models with database table schemas.

   CRUD
      Create, Read, Update, Delete - four basic operations of persistent storage.

   Window Function
      Function operating across rows related to the current row in Snowflake SQL.
